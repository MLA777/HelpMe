# HelpMe
团队名称：救救我

项目名称：混凝土加固的诺贝尔奖

痛点：人工巡检安全性差、裂缝发现不及时鲁棒性差、速度慢（人工测量裂缝指标速度慢）、精度低（人工目测）、速度慢（因裂缝形状导致速度慢）
问题定义：我们可以为检测人员创造速度更快、精度更高、适应性更好的裂缝实时检测方案，使他们能够对于桥梁裂缝实时监测更加精确、更加及时。

核心技术：数字图像相关测量（DIC技术）、图像配准、图像融合算法、基于深度学习的混凝土裂缝识别、分割、测量算法、多阶段裂纹识别框架、MU-net
